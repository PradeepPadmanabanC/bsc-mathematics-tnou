# BMSS-11C — Algebra

**Course Type:** Core  
**Course Mode:** Theory  

## Syllabus Topics

### Block 1 — Polynomial Equations
- [ ] Relationship between Roots and Coefficients
- [ ] Symmetric Function of the Roots
- [ ] Transformation of equations

### Block 2 — Roots & Series
- [ ] Increase or decrease the roots of a given equation
- [ ] Removal of term
- [ ] Descartes' rule of signs
- [ ] Approximate solutions of roots by Horner's method
- [ ] Cardan's method of solution of a cubic polynomial
- [ ] Summation of Series using Binomial, Exponential and Logarithmic series

### Block 3 — Matrices
- [ ] Symmetric & Skew Symmetric matrices
- [ ] Hermitian & Skew Hermitian matrices
- [ ] Orthogonal Matrices
- [ ] Eigen values & Eigen Vectors
- [ ] Cayley-Hamilton Theorem
- [ ] Similar matrices
- [ ] Diagonalization of a matrix

### Block 4 — Number Theory I
- [ ] Prime number
- [ ] Composite number
- [ ] Decomposition of a composite number as a product of primes (uniquely)
- [ ] Divisors of a positive integer n
- [ ] Euler function

### Block 5 — Number Theory II
- [ ] Congruence modulo n
- [ ] Highest power of a prime number p contained in n!
- [ ] Fermat's theorem
- [ ] Wilson's theorem
